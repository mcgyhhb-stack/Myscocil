# 🌟 موقعي الشخصي

موقع ويب شخصي احترافي يعرض مهاراتي ومشاريعي وخبراتي في تطوير الويب.

## 🚀 المميزات

- ✅ تصميم عصري وجذاب
- ✅ متجاوب مع جميع الأجهزة
- ✅ أنيميشن CSS3 متقدمة
- ✅ أداء عالي وسريع
- ✅ SEO محسن

## 🛠 التقنيات المستخدمة

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (ES6+)
- Font Awesome Icons

## 📱 الصفحات المتاحة

- الصفحة الرئيسية
- قسم عني
- قسم المهارات
- قسم المشاريع
- قسم الاتصال

## 🚀 النشر على Netlify

تم نشر الموقع على Netlify للحصول على:
- استضافة مجانية
- SSL تلقائي
- CDN عالمي
- نشر مستمر
